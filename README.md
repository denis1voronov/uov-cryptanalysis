This repository contains Python code to reproduce the experiments of partial key exposure attack on UOV schemes.
We use `cryptographic-estimators` package (https://github.com/Crypto-TII/CryptographicEstimators), licensed under Apache-2.0.

```bash
python -m venv .venv

# Windows
.venv\Scripts\activate

# Linux/macOS
source .venv/bin/activate

pip install -r requirements.txt
python estimations.py
```
